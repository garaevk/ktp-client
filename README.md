# КТП-генератор (клиентская версия)

Веб-приложение для создания календарно-тематического планирования, работающее полностью в браузере без серверной части.

## Возможности

- 📄 Загрузка и парсинг .docx файлов с таблицами КТП
- 📅 Загрузка и парсинг PDF календарного учебного графика
- 🗓️ Автоматическая генерация дат с учётом:
  - Четвертей/полугодий
  - Дней недели с настройкой количества уроков для каждого дня
  - Каникул
  - Праздничных дней
- 📝 Метаданные документа: школа, класс, предмет, уровень
- 💾 Создание и скачивание готового .docx документа с титульной страницей
- 🔒 Полная конфиденциальность — данные не покидают ваш компьютер

## Технологии

- **HTML5** — структура страницы
- **CSS3** — стилизация интерфейса
- **JavaScript (ES6+)** — логика приложения
- **JSZip** — парсинг .docx файлов (ZIP-архив)
- **pdf.js** — извлечение текста из PDF
- **docx.js** — создание .docx документов
- **FileSaver.js** — скачивание файлов

## Использование

### Локальный запуск

1. Откройте файл `index.html` в браузере:
   ```
   file:///путь/к/файлу/ktp_client/index.html
   ```

2. Или запустите локальный сервер:
   ```bash
   cd ktp_client
   python -m http.server 8000
   ```
   Затем откройте: http://localhost:8000

### Шаги работы

1. **Загрузка файлов**
   - Введите название школы
   - Загрузите файл КТП (.docx)
   - Опционально загрузите календарный учебный график (.pdf)
   - Укажите метаданные: класс, предмет, уровень

2. **Выбор таблицы**
   - Приложение автоматически найдёт таблицы в документе
   - Выберите таблицу с уроками

3. **Настройка дат**
   - Выберите дни недели для занятий и укажите количество уроков для каждого дня
   - Настройте четверти (автоматически из PDF или вручную)
   - Добавьте каникулы и праздничные дни

4. **Результат**
   - Просмотрите статистику генерации
   - Скачайте готовый .docx файл с титульной страницей

## Особенности

### Настройка количества уроков для каждого дня недели

В отличие от серверной версии, клиентская версия позволяет задать разное количество уроков для каждого дня недели. Например:
- Понедельник: 2 урока
- Среда: 1 урок
- Пятница: 3 урока

Это удобно для предметов с разной нагрузкой в разные дни.

### Титульная страница

Генерируемый документ включает полноценную титульную страницу с:
- Названием школы
- Предметом и уровнем
- Классом
- Датой создания

### Извлечение данных из .docx

Парсер извлекает из таблиц следующую информацию:
- Номер урока
- Тема
- Количество часов
- Контрольные работы
- Практические работы
- Ресурсы

## Ограничения

- Парсинг .docx требует определённой структуры таблицы (№ | Тема | Часы | Контрольные | Практические | Дата | Ресурсы)
- Парсинг PDF зависит от формата календарного графика
- Для работы требуется современный браузер с поддержкой ES6

## Отличия от серверной версии

| Функция | Серверная версия | Клиентская версия |
|---------|------------------|-------------------|
| Требуется сервер | Да | Нет |
| Конфиденциальность | Данные на сервере | Данные в браузере |
| Настройка уроков в день | Одно значение для всех дней | Отдельное значение для каждого дня |
| Скорость работы | Зависит от сервера | Мгновенно |
| Доступность | Требует интернет | Работает офлайн |
